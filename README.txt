游玩提示：

1.请提前在本地电脑中设置：“设置”->“系统”->“开发者选项”->“终端”->“Windows控制台主机”，以确保键盘的输入信息进入游戏窗口而不是程序黑框框。

2.游戏进行前，点击“设置->按键修改”，查看默认的游戏按键，修改方法为点击橙黄色方块后，再按下键盘按键即可（英文输入法），注意“方向键预设”和“字符自定义组合”的选择。

3.游戏进行中，请务必调整为英文输入法，防止游戏过程中按键识别失灵。

4.特色功能体验方式：先点击“玩法切换”，然后勾选“全新玩法”中的“植物大战方块（特色）”选项，点击“返回主页”，点击“开始游戏”后输入用户名“plantshow”，并选择第一个存档进行游玩即可，这个账号默认有足够多的分数给您享受“樱桃炸弹”、“寒冰菇”的植物力量！
